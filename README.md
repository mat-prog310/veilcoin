> Mine Simply. Transact Privately.

VeilCoin est une cryptomonnaie open-source axée sur la confidentialité et la simplicité de minage.

- 👥 **Gouvernance communautaire** - Pas de prémine, fair launch
- ⚡ **Rapide** - Blocs toutes les 2 minutes
